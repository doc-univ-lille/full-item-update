# English

Full-item-update is a Cloud-app made for Alma.
It allows a user with general administrator rights to update all the fields of an item listed in a CSV file.

A line in the CSV file is composed of the barcode (always the first column) and the different fields requiring a change.

To use this cloud-app **you need to have read the documentation on the configuration of the CSV file**

# Français

Full-item-update est une cloud-app Alma qui permet, en ayant le rôle d'administrateur général, de mettre à jour tous les champs des exemplaires listés dans un fichier CSV.

Pour chaque ligne du fichier CSV, on retrouve le code barre de l'exemplaire (toujours la première colonne) ainsi qu'une série de colonnes nécessitant un changement de valeur

Pour utiliser la cloud-app full-item-update, **il est impératif d'avoir lu la documentation sur la configuration du fichier CSV (trouvable dans le dossier README).**
